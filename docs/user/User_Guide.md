# User Guide

Welcome to Otto! Otto acts as a tireless employee for your small business, helping automate daily workflows securely.

## Getting Started
1. **The Resurrection:** Start by uploading a batch of paper invoices, handwritten ledgers, or WhatsApp chat exports via the `/resurrection` path. Otto will digitize your entire business in about 3 minutes.
2. **Approval Feed:** Monitor the home screen for pending actions. Otto initially requires your approval for every action (e.g., reordering supplies).
3. **The Autonomy Ladder:** Once you approve similar actions 3 times, Otto will ask for a "promotion". If granted, Otto will execute these actions autonomously.

## Safety Controls
You are always in control. Autonomously executed actions are capped, logged, reversible within a 1-hour window, and can be revoked entirely with a single switch.
