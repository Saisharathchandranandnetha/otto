# Administrator Guide

As an administrator of Otto, you can manage system settings, integrations, and high-level autonomy rules.

## Autonomous Workflow Agents Console
From the home screen, you can manage the "Autonomous Workflow Agents" console.
- **Stage Playbooks:** Pick a targeted industry (Education, Healthcare, Retail, etc.) and click "Stage" to create domain-specific action cards.
- **Revoking Trust:** If an autonomous workflow behaves unexpectedly, you can instantly revoke its trust toggle to return it to human-gated mode.

## API Keys
To use the full workflow orchestration API instead of local fallbacks, provide `OTTO_ENGINE_URL` and `OTTO_ENGINE_KEY` in your environment.
