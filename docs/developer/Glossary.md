# Glossary

- **Otto:** The AI operator that automates workflows for small businesses.
- **The Resurrection:** The process of converting offline raw materials (paper invoices, handwritten ledgers) into a fully digital business representation in minutes.
- **The Autonomy Ladder:** The mechanism by which Otto gains trust. Actions start human-gated, and after repeated approvals, Otto can execute them autonomously within limits.
- **Domain Engine:** Playbooks designed for specific industries (e.g., HR, Healthcare, Legal).
- **Shoebox Kit:** A collection of test documents (invoices, ledgers) used for 'The Resurrection'.
